Mastermind Game Rules

Objective of the Game

To break the secret color code in the least number of steps(guesses).
Computers goal as a code maker is to set a mystery code so cunning that it will keep you guessing for as long as possible.
As a Decoder: you must break the secret code in the fewest number of guesses.

Game Play

1. Decoder(you) can begin guessing, trying to duplicate the exact colors and positions of the hidden code colors. Each guess is made by choosing colors on the players row with the send button.
2. After every guess, the Codemaker(computer) must inform the Decoder(you) of their progress using four smaller dots on the right side of each lower row(players response history)

Codemaker(computer) reply explained

Codemaker(computer) uses the right side of each lower row(players response history) to provide a feedback on each player's guess

Green Indicator

Green small dots displays the number of players color guesses that are the correct colors in the correct positions.

White Indicator

White small dots displays the number of players color guesses that are the correct colors but in the wrong positions.

End of the Game

If the Decoder does not crack the code in chosen number of attempts or if timer run out, the game is over.
